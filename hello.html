Kumpulan command git

  1. git commit -am "Add a heading": untuk melakukan cloning all message
  2. git status: untuk mengecek status file saat ini
  3. git push: melakukan push/clone dari local computer ke web github
  4. git clone https://github.com/bartan29/basic-web-programming.git: untuk melakukan clone dari github ke local computer
  5. git reset
    • git reset --hard (commit)
    • git reset --hard origin/master
  6. git pull
  7. git log
  8. git branch
    • git checkout -b style: untuk create new branch bernama style
    • git checkout master: untuk pindah branch
